JQuery DataTable Row Reordering - Integration with the Row Grouping plugin

You can reorder rows even if they are partitioned in the groups. The only constraint it that you cannot move rows between different groups (if you need this you will need to edit row first to change its group, and then reorder it). In the example below try to drag and drop rows withing the same group to change their positions.
