JQuery DataTable Row Reordering

To reorder rows in the table, just drag them and drop them to new position. Row Reordering plugin will adjust positions in the indexing columns. Note that plugin forces reordering by indexing column and this cannot be changed. if you want to use this pluign you will not be able to sort rows by other columns because positions of the rows in the table will be mixed.

Live example

			/* 
			 * Example init
			 */
			$(document).ready(function(){
				 $('#example')
				 	.dataTable()
				 	.rowReordering();
			});
